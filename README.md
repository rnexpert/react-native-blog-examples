## 视频播放器功能

- [x] 支持本地和远程视频地址
- [x] 支持播放和暂停
- [x] 支持横竖屏切换
- [x] 横屏显示视频标题
- [x] 静音功能
- [x] 支持视频选集切换
- [x] 横屏清晰度切换
- [x] 横屏展示分享
- [x] 调整音量
- [x] 调整播放速率，支持多倍速播放
- [x] 适配iPhone X
